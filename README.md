# 🔐 Secure Closed Group Communication System.

## 🎯 Objective

The main objective of this project is to design and develop a secure backend system for user communication using modern web technologies. The system aims to provide a structured and scalable platform for user registration and future secure message exchange.

This project focuses on:

* Building RESTful APIs for user management
* Storing and retrieving user data securely from a database
* Creating a strong backend foundation for secure communication systems

---

## 📌 Project Description

SecureComm is a backend application developed using Java and Spring Boot.
It allows users to register and stores their details in a MySQL database.
The system follows a layered architecture consisting of controller, service, repository, and model layers.

---

## ⚙️ Technologies Used

* Java
* Spring Boot
* Spring Data JPA
* MySQL
* Maven

---

## 🔄 Working Procedure

1. The user sends a request (JSON data) to the backend API.
2. The request is received by the controller layer.
3. The controller forwards the request to the service layer.
4. The service layer processes the data and applies business logic.
5. The repository layer interacts with the database.
6. Data is stored in the MySQL database.
7. The response is sent back to the user.

---

## 🧩 System Architecture

* Controller Layer → Handles API requests
* Service Layer → Contains business logic
* Repository Layer → Handles database operations
* Model Layer → Represents data structure
